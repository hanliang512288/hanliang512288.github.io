Set Vim to send and receive Email

Contents

  1. 1. Vamil
  2. 2. Installation Issues

Using vim to send and receive Email seems to be a geek thing for most of the Vimers. In this post, I will introduce how to deal with Emails with vim.

Vamil

Vmail is a plugin based on vim, which make vim send and reveive Email. The installation is simple, but for Ubuntu, it need some dependencies installation and configuration.

Vmail is a ruby gem package, so we need to assure your system is installed and configured with ruby. Please refere this blog [How to install Ruby, Rvm, Ruby on Rails on ubuntu] ().

Other installation and manipulation notes of Vmail, please refer to Vmail’s github.

Installation Issues

Normally, after finishing installing Vmail, you type “Vmail” command, the system will directly log into your Email. But sometimes the system will indicate that you need to install “w3m” or other packages. These packages are mainly about displaying some hyperlink or images in the command line tools. Just use the sudo apt-get w3m or install other dependent packages in the terminal.
