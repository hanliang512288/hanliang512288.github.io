1. Datastructure

1.1 Primitive Data Structures

Array: a sequence of n items of the same data type that are stored contiguously in computer memeory and made accessible by speicfying a value of the array’s index.

Bubble sort: sometimes referred to as sinking sort, is a simple sorting algorithm that repeatedly steps through the list to be sorted, compares each pair of adjacent items and swaps them if they are in the wrong order. The pass through the list is repeated until no swaps are needed, which indicates that the list is sorted

Insertion Sort: Each iteration, insertion sort removes one element from the input data, finds the location it belongs within the sorted list, and inserts it there. It repeats until no input elements remain.
