1. Java Vitual Machine

High-level language: human can read, write and understand

Machine language: computer can understand

Low-levle language: machine language or any language similar to machine language

Compiler: a program that translates a high-level language into low-level language

  • Most programming langauge : High-level -> [compiler] -> low-level language (vary on machines)

  • Java : High-level -> [Compiler] -> byte code (for JVM)-> [Interpreter]-> machine language
